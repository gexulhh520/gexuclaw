核心目标：

不推翻现有 plan / step / inputRefIds / TaskEnvelope 设计，只补齐“上一步子 Agent 产物自动传给下一步子 Agent”的链路，并优化子 Agent prompt 拼接。

修改任务说明

请基于当前仓库 agent-platform-node 修改 orchestration 相关代码。

当前已有结构：

MainAgent 负责生成 MainDecision
ExecutionPlanCompiler 把 MainDecision.plan.steps 编译成 ExecutionPlan
executePlanAsync 顺序执行每个 step
TaskEnvelopeBuilder 根据 step 构建 TaskEnvelope
AgentRuntime 执行子 Agent
AgentResultBuilder 从 run steps / artifacts 生成 AgentResult

当前主要问题：

inputRefIds 只能引用主 Agent 规划前已经存在的 refs，不能引用执行过程中刚产生的 run / artifact / file。
上一个子 Agent 执行完之后，虽然生成了 run / artifact / touchedResources，但下一个 step 不会自动拿到。
TaskEnvelopeRenderer 同时渲染 Selected Context 和 Ledger / Artifacts / Files，存在重复。
ledgerSlices 当前只显示 refId/status，没有有效摘要策略。
子 Agent prompt 中仍然可能包含 Original User Message，容易导致子 Agent 越权理解全局任务。
不应该默认把上一个 Agent 的所有 run steps 都传给下一个 Agent，只在失败、验证、诊断、解释 trace 时传关键 steps。

当前代码中，TaskEnvelopeBuilder 会从 step.inputRefIds 展开 refs，并根据 ref.kind 构建 ledgerSlices / artifacts / files，最后同时放入 selectedContext.refs 和详情切片。这个是重复来源。
TaskEnvelopeRenderer 当前会渲染 Original User Message、Selected Context、Ledger Slices、Artifacts、Files 等区块。

新增 helper：

function selectLedgerSteps(
  steps: RuntimeStepTrace[],
  mode: "summary" | "critical_steps" | "full",
  maxSteps: number
): RuntimeStepTrace[] {
  if (mode === "summary") return [];

  if (mode === "full") {
    return steps.slice(-maxSteps);
  }

  return steps
    .filter((s) =>
      s.stepType === "tool_end" ||
      s.stepType === "error" ||
      s.toolStatus === "failed" ||
      s.toolStatus === "error"
    )
    .slice(-maxSteps);
}

注意：需要从类型文件引入 RuntimeStepTrace。

建议新增文件：

内容：

import { eq } from "drizzle-orm";
import { db } from "../../db/client.js";
import { agentRuns, agentRunSteps, agentArtifacts } from "../../db/schema.js";
import { jsonParse } from "../../shared/json.js";
import type { ContextRef } from "./orchestration.types.js";
import type { LedgerSlice, ArtifactSlice, FileSlice } from "./task-envelope.types.js";

export async function hydrateTaskEnvelopeContext(refs: ContextRef[]): Promise<{
  ledgerSlices: LedgerSlice[];
  artifacts: ArtifactSlice[];
  files: FileSlice[];
}> {
  const ledgerSlices: LedgerSlice[] = [];
  const artifacts: ArtifactSlice[] = [];
  const files: FileSlice[] = [];

  for (const ref of refs) {
    if (ref.kind === "run") {
      const runUid = ref.source?.runUid || ref.source?.uid || "";
      if (!runUid) continue;

      const [run] = await db
        .select()
        .from(agentRuns)
        .where(eq(agentRuns.runUid, runUid))
        .limit(1);

      if (!run) continue;

      const steps = await db
        .select()
        .from(agentRunSteps)
        .where(eq(agentRunSteps.runId, run.id))
        .orderBy(agentRunSteps.stepIndex);

      ledgerSlices.push({
        refId: ref.refId,
        runUid,
        agentUid: "",
        status: run.status || ref.status || "unknown",
        steps: steps.map((s) => ({
          stepIndex: s.stepIndex,
          stepType: s.stepType,
          content: s.content ?? undefined,
          toolName: s.toolName ?? undefined,
          toolCallId: s.toolCallId ?? undefined,
          toolStatus: s.toolStatus ?? undefined,
          inputJson: jsonParse(s.inputJson, {}),
          outputJson: jsonParse(s.outputJson, {}),
          metadataJson: jsonParse(s.metadataJson, {}),
          createdAt: s.createdAt,
        })),
      });
    }

    if (ref.kind === "artifact") {
      const artifactUid = ref.source?.uid || "";
      if (!artifactUid) continue;

      const [artifact] = await db
        .select()
        .from(agentArtifacts)
        .where(eq(agentArtifacts.artifactUid, artifactUid))
        .limit(1);

      if (!artifact) continue;

      artifacts.push({
        refId: ref.refId,
        artifactUid: artifact.artifactUid,
        title: artifact.title,
        artifactType: artifact.artifactType,
        artifactRole: artifact.artifactRole ?? undefined,
        summary: artifact.summary ?? ref.summary,
        contentText: artifact.contentText ?? undefined,
        contentJson: artifact.contentJson ? jsonParse(artifact.contentJson, undefined) : undefined,
      });
    }

    if (ref.kind === "file") {
      files.push({
        refId: ref.refId,
        uri: ref.source?.uri || "",
        path: ref.source?.uri || "",
        lastKnownStatus:
          ref.status === "write_failed"
            ? "failed"
            : ref.status === "ok"
              ? "success"
              : ref.status === "unverified"
                ? "unverified"
                : "unknown",
        summary: ref.summary,
      });
    }
  }

  return { ledgerSlices, artifacts, files };
}

注意：字段名要和你的 Drizzle schema 实际字段保持一致。如果 agentArtifacts.summary 或 contentJson 字段不存在，请按实际 schema 调整。
